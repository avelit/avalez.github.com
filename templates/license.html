You can request license here to enable new features, like Timesheet Configuration, Timesheet Holidays, Timesheet Email and Worked Time Chart in JIRA OnDemand.

This is temporary solution until Atlassian enables licensing for plugin for JIRA OnDemand. It has no variation by number of users in JIRA, so price is the same for all size companies. And only payments via PayPal are supported with no any kind of invoice beside standard PayPal confirmation are possible.

Once you pay chosen amount, you will shortly receive license code to email address you used for PayPal payment.

Please, chose period of license validity, specify name of your Organization (will be displayed in License) and click Pay Now below to be redirected to payment page. Should you have any questions, please do not hesitate to write an email.
